## Mappings [.wezterm.lua](https://github.com/sk1t0n/dotfiles/blob/master/home/anton/.wezterm.lua#L16)

| key                | description                   |
|--------------------|-------------------------------|
| ALT + Enter        | toggle full screen mode       |
| CTRL + SHIFT + C   | copy to the clipboard         |
| CTRL + SHIFT + V   | paste from the clipboard      |
| CTRL + Tab         | go to the next tab            |
| CTRL + SHIFT + Tab | go to the previous tab        |
| CTRL + ALT + n     | create a new tab              |
| CTRL + ALT + c     | close the current pane or tab |
| CTRL + ALT + ;     | create a new vertical pane    |
| CTRL + ALT + '     | create a new horizontal pane  |
| CTRL + ALT + 1     | switch to the left pane       |
| CTRL + ALT + 2     | switch to the bottom pane     |
| CTRL + ALT + 3     | switch to the top pane        |
| CTRL + ALT + 4     | switch to the right pane      |
